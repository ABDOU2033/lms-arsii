🚀 DÉMARRAGE RAPIDE

  1️⃣  Installation de dépendances
      $ composer install && npm install

  2️⃣  Configuration environnement
      $ cp .env.example .env
      $ php artisan key:generate

  3️⃣  Base de données
      $ php artisan migrate

  4️⃣  Lancer les serveurs
      Terminal 1: $ php artisan serve --host=127.0.0.1 --port=8000
      Terminal 2: $ npm run dev

  5️⃣  Accéder à l'application
      🌐 http://127.0.0.1:8000

📚 GUIDE COMPLET DU SYSTÈME DE QUIZ LMS
=========================================

✅ FLUX COMPLET D'UN ÉTUDIANT POUR RÉPONDRE À UN QUIZ

1️⃣ ÉTUDIANT SE CONNECTE
   - Email: etudiant@lms.com
   - Mot de passe: Karim@2024
   - Accès: http://127.0.0.1:8000/login

2️⃣ ACCÈS AUX QUIZZES
   - Dashboard → "Mes Cours"
   - Sélectionne un cours: ex "HTML5 et CSS3"
   - Voit 2 sections:
     ✓ LEÇONS (gauche)
     ✓ QUIZ (droite)

3️⃣ AFFICHAGE DES QUIZZES
   Pour chaque quiz, l'étudiant voit:
   - 📌 Titre du Quiz: "Quiz CSS3 Avancé"
   - ⏱ Durée: 20 minutes
   - 📊 Note Max: 15 points
   - 📝 Nombre de questions: 3
   - 🎯 Bouton "Commencer"

   EXEMPLE:
   ┌─────────────────────────────────────────────┐
   │ Quiz CSS3 Avancé                    Commencer│
   │ ⏱ 20 min • 📊 15 pts                         │
   │ 3 questions                                  │
   └─────────────────────────────────────────────┘

4️⃣ CLIC SUR "COMMENCER"
   Route: /etudiant/quiz/{quiz_id}
   Action: EtudiantController@showQuiz()
   
   ✓ Charge le quiz avec ses 3 questions
   ✓ Charge les choix_reponses pour chaque question
   ✓ Vérifie si l'étudiant a déjà répondu

6️⃣ ÉTUDIANT RÉPOND AUX 3 QUESTIONS
   
   ✓ Les réponses sont stockées en tempo
   ✓ Champ de formulaire: name="reponses[{question_id}]"
   ✓ Required: tous les champs doivent être remplis

   EXEMPLE DE SOUMISSION:
   POST /etudiant/quiz/5/submit
   Data:
   - reponses[9] = "Vrai"        (Q1)
   - reponses[10] = "Vrai"       (Q2)
   - reponses[11] = "justify-content et align-items" (Q3)

7️⃣ CLIC SUR "SOUMETTRE LE QUIZ"
   Route: POST /etudiant/quiz/{quiz_id}/submit
   Action: EtudiantController@submitQuiz()
   
   PROCESSUS INTERNE:
   ✓ Valide que tous les champs sont remplis
   ✓ Vérifie que l'étudiant n'a pas déjà soumis
   ✓ Crée 3 records Reponse:
     - etudiant_id: 1
     - question_id: 9
     - contenu: "Vrai"
     - est_correcte: false (par défaut)
     - score_obtenu: 0 (par défaut)
   
   ✓ Appelle Quiz::corrigerAutomatiquement() pour chaque réponse:
     - Si type QCM/Vrai-Faux:
       * Cherche ChoixReponse avec:
         - contenu = réponse de l'étudiant
         - est_correcte = true
       * Si trouvé:
         - Reponse.est_correcte = true
         - Reponse.score_obtenu = question.points
       * Si pas trouvé:
         - Reponse.est_correcte = false
         - Reponse.score_obtenu = 0
   
   ✓ Calcule le score total:
     - Quiz::calculerScore($etudiant)
     - Somme les score_obtenu
     - Limite à quiz.note_max
